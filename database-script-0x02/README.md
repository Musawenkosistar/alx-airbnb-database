# Sample Data for Airbnb-like Database

This SQL script populates the database with realistic sample data for testing purposes:

- **Users**: 3 sample users
- **Properties**: 4 properties owned by different users
- **Bookings**: Sample bookings reflecting real usage
- **Payments**: Payments associated with bookings, some completed, some pending

The data allows testing queries, joins, and reporting functionality for the Airbnb-like application.
